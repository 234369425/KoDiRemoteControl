# KoDi简易遥控器 #
  KoDi原有界面操作过于复杂，不适合老人、孩子使用，基于
	[KoDi RPC API v8](https://kodi.wiki/view/JSON-RPC_API/v9)
	实现了一个简易遥控器

- 支持配置地址
- 支持音量控制
- PRV频道直接选择
- KoDi pvr地址: http://kc.eastapple.com/playlist.m3u8


[跳转试用地址](http://kc.eastapple.com "在线试用地址")
